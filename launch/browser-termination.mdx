---
title: "Termination"
---

Kernel makes it possible to persist and reuse browser sessions. Browsers not marked for persistence are automatically terminated after your [action](/build/develop) ends, or the you call `browser.close()` on a [remote browser](/build/remote-browsers).

To terminate a persisted browser, delete it using the browser ID that you specified when creating the browser [with persistence](/launch/browser-persistence):

```bash
kernel browser delete <persistent_browser_id>
```

You can also do this by making a `DELETE` request to Kernel's API:
<CodeGroup>

```typescript Typescript/Javascript
const response = await fetch(`https://api.onkernel.com/browsers?persistent_id=${browserId}`, {
    method: 'DELETE',
    headers: {
        'Authorization': `Bearer ${process.env.KERNEL_API_KEY}`,
        'Content-Type': 'application/json'
    }
});
```
```python Python
response = requests.delete(
    f"https://api.onkernel.com/browsers?persistent_id={browser_id}",
    headers={
        'Authorization': f"Bearer {os.environ.get('KERNEL_API_KEY')}",
        'Content-Type': 'application/json'
    }
)
```
</CodeGroup>

Or via the SDK:
<CodeGroup>
```typescript Typescript/Javascript
import Kernel from '@onkernel/sdk';
const kernel = new Kernel();

await kernel.browsers.delete({ persistent_id: 'persistent_id' });
```

```python Python
from kernel import Kernel
client = Kernel()

client.browsers.delete(
    persistent_id="persistent_id",
)
```
</CodeGroup>

See the [CLI reference](/reference/cli) for full details.
